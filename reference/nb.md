# Norwegian characters in unicode

Norwegian characters in unicode

## Usage

``` r
nb
```

## Format

An object of class `list` of length 6.

## Examples

``` r
print(csdata::nb)
#> $AA
#> [1] "Å"
#> 
#> $aa
#> [1] "å"
#> 
#> $OE
#> [1] "Ø"
#> 
#> $oe
#> [1] "ø"
#> 
#> $AE
#> [1] "Æ"
#> 
#> $ae
#> [1] "æ"
#> 
```
